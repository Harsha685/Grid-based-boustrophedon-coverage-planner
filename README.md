# Boustrophedon Coverage Planner

A grid-based Boustrophedon coverage planner written in Python using NumPy and Matplotlib.

## Features

- Grid-based cellular decomposition
- Boustrophedon sweeping
- Pathfinding between disconnected regions
- Coverage visualization

## Installation

```bash
poetry install
```

## Run

```bash
poetry run python -m path_coverage_algorithm
```
